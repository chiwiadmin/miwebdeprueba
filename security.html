Seguridad

Seguridad como base de todo lo que hacemos

La confianza es la base de nuestro trabajo con infraestructura crítica y ciberseguridad.

Privacidad

Tratamos la información de nuestros clientes con estricta confidencialidad. Solo accedemos a los sistemas y datos necesarios para ejecutar el proyecto acordado, y bajo el alcance definido con cada cliente.

Protección de datos

Aplicamos controles técnicos y organizativos razonables para resguardar la información con la que trabajamos, incluyendo control de acceso, cifrado en tránsito cuando corresponde y minimización de datos.

Buenas prácticas

Seguimos lineamientos reconocidos de la industria en nuestras implementaciones y auditorías, adaptando cada recomendación al contexto real del cliente en lugar de aplicar checklists genéricas.

Divulgación responsable

Si detectás una vulnerabilidad relacionada con nuestros sistemas o sitio, te pedimos reportarla de forma responsable antes de hacerla pública, para poder evaluarla y resolverla.

Podés escribirnos a contacto@lulupacho.org
